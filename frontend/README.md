# Vibeo — Real-Time Video Conferencing App

A WebRTC-based video conferencing web app with real-time peer-to-peer video/audio, 
in-call chat, and screen sharing.

## Tech Stack
- **Frontend:** React, Material UI, Socket.IO Client
- **Backend:** Node.js, Express, Socket.IO, MongoDB (Mongoose)
- **Real-time:** WebRTC (peer-to-peer), Socket.IO (signaling)

## Features
- Peer-to-peer video & audio calls via WebRTC
- Real-time in-call chat
- Screen sharing
- User authentication & meeting history (JWT-based)

## Run Locally

**Backend**
```bash
cd backend
npm install
npm run dev
```

**Frontend**
```bash
cd frontend
npm install
npm start
```
